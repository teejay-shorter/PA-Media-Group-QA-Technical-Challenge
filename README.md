## Getting Started

### Dependencies

- npm
- Mocha
- Chai
- Supertest
- Babel

### Installing

- Clone the repo
- Navigate to the root dir
- Open the terminal
- Run the following command:

```
npm init
```

### Executing program

- Once the project has been intialised, run the following command:

```
npm test
```
